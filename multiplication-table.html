Diese Seite ist um das Thema Rechnen im Besonderen der Multiplikation aufgebaut. Ich benutzte das JavaScript Framework React zum Erstellen der Seite.


Aufbau der Seite

Wenn die Seite geladen wird, erscheint zuerst der Setting Bildschirm, auf dem Einstellungen zur Abfrage eingeben werden können. Nach dem Klick auf Create Table, erscheint eine Tabelle über alle ausgewählten Reihen. Auf diesem Bildschirm kann die Abfrage gestartet oder wieder auf den Einstellungs-Bildschirm zurückgekehrt werden. Wird die Abfrage gestartet, erscheinen der ersten zwei Zahlen. Und drei Antwort-Möglichkeiten. Darunter ein Reset-Button und ein Button zum Zurückkehren auf die Einstellungsseite.


Schwierigkeiten

Die Zeitmessung war eine der grössten Schwierigkeiten. In der React-Komponente SetExercise.js braucht es für die Zeitmessung im Ganzen fünf Methoden.


Spezielles

In der App.js-Komponente wendete ich eine spezielle Technik an, die zum Eindruck führt, dass die Webseite nicht nur eine Ein-Seiten-Webseite ist. Im Konstruktor von App.js legte ich im Objekt state drei Schlüssel mit den booleschen Werten true oder false an. Mit diesen Schaltern kann nun der Eindruck einer Mehr-Seiten-Webseite erweckt werden.


Aussichten

Auch diese Seite möchte ich mit einer API verbinden. Die Zeit-Werte möchte ich in einer Datenbank ablegen. Und für eine Scorer-Liste benutzen.
